# WhatsApp Sender (Demo)

## Описание проекта

Демонстрационная программа для отправки сообщений через WhatsApp.

Проект выполнен в рамках тестового задания и соответствует ограничению:

* массовая рассылка по чужим номерам не выполняется;
* реализован режим dry-run (по умолчанию);
* live-режим используется только для отправки 1–2 сообщений на собственные или тестовые номера.

## Функциональность

* Чтение контактов из CSV-файла;
* Режим dry-run (имитация отправки);
* Режим live (демонстрационная отправка);
* Логирование действий в файл;
* Обработка ошибок (некорректный номер и др.).

## Структура входных данных

CSV-файл должен содержать следующие столбцы:

* номер
* сообщение

Пример:

номер,сообщение
77474008762,Good night!
77055814012,Good morning!

## Используемые библиотеки

* pandas
* pywhatkit
* openpyxl
